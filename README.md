# kuailv-legal-content-agent-skill
快律：一个面向法律短视频内容生产、法律依据检索、长链法律推理和合规审查的多 Agent Skill 原型。
## 项目介绍

「快律」是一个面向法律内容生产场景的 AI Agent Skill 原型项目，目标是将法律材料理解、法律依据检索、法律推理、短视频内容生成、合规审查和多平台分发整合为一套自动化工作流。它并不是一个简单的文案生成工具，而是围绕法律内容创作中的真实流程进行拆解，通过多 Agent 协作完成从“输入法律材料”到“输出可发布内容包”的完整闭环。

在实际法律内容生产中，创作者经常需要处理裁判文书、热点新闻、咨询记录、法条原文、司法解释和典型案例等长文本材料。这些材料通常结构复杂、信息密度高，既包含事实经过，也包含争议焦点、法律关系、证据问题和责任判断。对于法律自媒体、律师助理、普法账号运营者和法律营销团队而言，人工完成这些工作往往需要经过资料阅读、事实整理、法条检索、案例比对、法律分析、脚本改写、标题设计和合规检查等多个步骤，耗时较长，且质量高度依赖个人经验。「快律」希望通过 AI Agent 的方式，把这些分散的人工流程转化为可复用、可扩展、可评估的自动化工作流。

「快律」的核心能力是：用户输入一个法律热点、裁判文书摘要、真实咨询记录、法条片段或新闻材料后，系统能够自动完成案件事实梳理、争议焦点提取、法律依据匹配、法律分析结论生成、短视频口播稿创作、标题封面文案生成、平台标签推荐、评论区互动话术生成，以及发布前的合规风险审查。最终输出的不只是单篇文案，而是一套完整的法律内容生产包，适合用于抖音、小红书、视频号、公众号等不同内容平台。

项目采用多 Agent 协作架构。输入解析 Agent 负责从原始材料中提取主体、时间线、行为、证据、争议焦点和潜在法律关系；法律研究 Agent 负责通过 RAG 检索法律法规、司法解释、典型案例和裁判规则；法律推理 Agent 负责基于事实和依据进行长链推理，判断责任划分、举证难点、法律风险和不确定因素；内容转译 Agent 负责将专业法律表达转化为普通用户能够理解的语言；短视频创作 Agent 负责生成 60 秒口播稿、30 秒精简版、标题、封面文案和标签；合规审查 Agent 负责检查夸大承诺、误导性法律建议、隐私泄露、绝对化判断、敏感表达和平台违规风险；质量评估 Agent 则负责对最终输出进行评分，检查法律依据是否充分、逻辑链是否闭合、表达是否清晰、传播钩子是否明确、合规风险是否已处理。

从技术设计上看，「快律」强调长上下文处理、RAG 检索、多阶段推理和输出质量控制。项目后续计划接入法律知识库，包括常用法律法规、司法解释、典型案例、普法脚本模板和平台内容合规规则，使系统在生成内容前能够先检索，再分析，再输出，减少幻觉和错误结论。同时，项目也会加入人工审核反馈、版本对比、批量生成、内容质量评分和多平台适配能力，使其逐步从单次内容生成工具升级为面向法律内容团队的 AI 工作流系统。

该项目具有较高的 token 使用需求。单次任务通常需要处理较长的法律文本材料，并经过多轮抽取、检索、推理、生成、改写、审查和评分。一个完整任务可能涉及多个 Agent 连续调用模型，并需要保留较长上下文以避免前后结论冲突。后续接入法规库、案例库和平台规则库后，还会产生更高频的 RAG 检索、上下文拼接、批量测试和输出评估需求。因此，「快律」适合作为高 token 消耗、多 Agent 协作和法律垂直场景落地的原型项目。
