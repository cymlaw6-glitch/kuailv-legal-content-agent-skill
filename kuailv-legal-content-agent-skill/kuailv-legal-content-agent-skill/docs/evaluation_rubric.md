# 输出质量评估标准

## 1. 法律依据完整性

| 分数 | 标准 |
|---|---|
| 0-3 | 缺少法律依据或依据明显错误 |
| 4-6 | 有基础依据，但适用边界不清 |
| 7-8 | 依据较完整，能支持主要结论 |
| 9-10 | 依据充分，且标明不确定性和适用条件 |

## 2. 逻辑链闭合度

检查事实、争议焦点、法律依据和结论之间是否一致。

## 3. 表达通俗度

检查是否把专业法律分析转化为普通用户能理解的表达。

## 4. 传播钩子强度

检查短视频脚本是否具备开头冲突、节奏推进和结尾提醒。

## 5. 合规安全性

检查是否存在：

- 绝对化结论；
- 夸大承诺；
- 误导性建议；
- 隐私泄露；
- 诽谤风险；
- 标题党风险；
- 平台违规风险。

## 6. 平台适配度

检查内容是否适合目标平台，例如抖音、小红书、视频号或公众号。
